## 💡 КЛЮЧОВІ НАВЧЕНІ НАВИЧКИ

1. **ARIA атрибути**
   - aria-expanded для комунікації стану
   - aria-controls для зв'язування кнопки та панелі
   - aria-labelledby для присвоєння назв

2. **Семантична HTML**
   - Нативні елементи мають вбудовану доступність
   - &lt;button&gt; забезпечує табуляцію та активацію
   - &lt;h3&gt; надає структурну інформацію

3. **CSS селектори**
   - Керування видимістю на основі ARIA атрибутів
   - Динамічні селектори без додаткових класів

4. **Клавіатурна навігація**
   - Обробка подій keydown
   - Циклічна навігація (модульна арифметика)
   - Пропускання подій браузера (preventDefault)

5. **DevTools для аудиту**
   - Дерево доступності показує те, що бачать екранні диктори
   - Перевірка ARIA атрибутів та їхніх значень
   - Source Order Viewer для перевірки логіки табуляції
